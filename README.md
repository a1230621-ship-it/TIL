# TIL (Today I Learned)

## 소개
이 프로젝트는 매일 새로 배운 기술, 프로그래밍 개념, AI 활용법 및 공부한 내용을 자유롭게 기록하는 개인 학습 저장소입니다.

## 주요 기능
- **일일 학습 기록:** 파이썬, 데이터 분석, AI 프롬프트 등 그날 배운 핵심 내용 정리
- **트러블슈팅 일지:** 코딩 중 발생한 에러와 해결 과정을 기록하여 복기
- **지식 아카이브:** 나중에 다시 찾아볼 수 있는 나만의 기술 사전 구축

## 사용 방법
1. 공부한 주제에 맞는 폴더 또는 마크다운(`.md`) 파일을 생성합니다.
2. 당일 학습한 핵심 요약과 코드를 정리하여 커밋(Commit)합니다.
